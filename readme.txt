# Jogo em Pygame Zero

Este é um jogo simples de plataforma 2D desenvolvido com Pygame Zero.

## Como Executar

1. Instale as dependências: pip install -r requirements.txt
2. Execute o jogo: python main.py

## Descrição

O jogo apresenta um personagem jogador que pode se mover para esquerda/direita, pular e evitar inimigos. O objetivo é alcançar a bandeira para vencer. Há menus, opções de som, etc.

## Controles

- Setas esquerda/direita: mover
- Seta para cima: pular
- Mouse: interagir com o menu
